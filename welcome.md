## Getting Started

**Step 1:** Make sure you have Qwen Code installed.

If not, download it here: [https://qwen.ai](https://qwen.ai)

**Step 2:** Run the installation script.

On Windows (PowerShell):
```powershell
powershell -ExecutionPolicy Bypass -File "path\to\install.ps1"
```

Or manually copy the folder:
```powershell
Copy-Item -Path "path\to\agentlify-course" -Destination "$env:USERPROFILE\.qwen\courses\agentlify" -Recurse
```

**Step 3:** Start the course.

In Qwen Code, type:
```
/agentlify:welcome
```

Or jump straight to Lesson 1:
```
/agentlify:lesson-1
```
