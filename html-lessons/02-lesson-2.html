Build Your First Skill

~8 minutes
Goal: Build a custom skill
Win: YOUR command that does work

The Meta Moment

Here's something that'll blow your mind — you've been using skills this entire time.

You know how you typed /agentlify:lesson-1 to start the last lesson? That's a skill.

The welcome page? Also a skill.

Every time you typed one of those slash commands, Qwen loaded up a text file with instructions and followed them. That's it. That's the whole thing.

Skills are just text files that tell Qwen what to do. Think of it like an SOP for your assistant. You know how you'd write out "here's how I want you to handle new client inquiries" for a team member? Same thing. Except instead of a Google Doc they might read once, it's a command that Qwen follows perfectly every single time.

What's Inside a Skill?

Let me show you what a skill actually looks like on the inside. There are only 3 parts:

The 3 Parts of a Skill

Part 1 — The Frontmatter

That stuff between the --- dashes at the top. This is like the name tag. It has a description that tells Qwen what the skill does. This is what shows up when you type / in Qwen Code — it's the little preview text.

Part 2 — The Instructions

The main body. This is just plain English telling Qwen what to do. "Show this banner. Say this intro. Present these options." No code. No programming. Just words.

Part 3 — The Rules

The guardrails at the bottom. "Always speak in first person. Bold key phrases. Every sentence on its own line." Think of them as the standards you'd set for any good team member.

Here's the key takeaway: It's a text file. You write instructions in plain English and Qwen follows them. No code. No API. No technical skills required. Like writing an SOP for the best assistant you've ever had.

Step 1: Choose Your Skill

Now the fun part — we're building you a custom skill right now. Based on your coaching or consulting business, here are some ideas:

Skill Ideas

  • Session Prep — Creates a prep sheet for each client session
  • Discovery Questions — Generates smart intake questions for prospects
  • Client Welcome — Creates a personalized onboarding message
  • Content Ideas — Generates a week of content ideas
  • Workshop Outline — Creates a workshop structure from a topic
  • Client Check-In — Drafts personalized weekly check-ins

Or if you have a different idea — something you do ALL the time that you'd love to automate — build that instead.

Run the Command

Open Qwen Code and type:

/agentlify:lesson-2

Qwen will walk you through building your skill step by step.

Step 2: Run Your Skill

Once your skill is built, test it immediately. Type your new command:

/my-[skill-name]

Watch it do its thing. That skill is YOURS. You designed it. You wrote the instructions. You told Qwen exactly what to do and it did it.

That's not a template I gave you. That's YOUR creation. And here's the beautiful part — that skill is saved on your machine forever. You can run it anytime. Edit it anytime. Make it better anytime.

Bonus Unlocked: Skill Builder

You just built one skill with help. But what if you could build skills on your own? Without anyone walking you through it?

I'm giving you the Skill Builder. It's a skill that builds other skills. I know that sounds meta, but that's what makes it useful. You tell it what you need, and it creates the whole skill file — frontmatter, instructions, rules, everything. Saved and ready to use.

Location: ~/.qwen/skills/skill-builder.md

Want a skill for client emails? Build it. Want a skill for proposals? Build it. Want a skill for workshop outlines? BUILD IT.
