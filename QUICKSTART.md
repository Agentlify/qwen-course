## Installation (2 minutes)

### Windows

1. **Download** the `qwen-agents-course` folder

2. **Open PowerShell** and run:
   ```powershell
   powershell -ExecutionPolicy Bypass -File "C:\path\to\qwen-agents-course\install.ps1"
   ```

3. **Done.** The course is installed.

### Mac/Linux

1. **Download** the `qwen-agents-course` folder

2. **Open Terminal** and run:
   ```bash
   cp -r /path/to/qwen-agents-course ~/.qwen/courses/lfg
   ```

3. **Done.** The course is installed.

## Start the Course

Open Qwen Code and type:
