## How to Start (For Students)

**Step 1:** Make sure you have Qwen Code installed.

**Step 2:** Run the installation script:

```powershell
# Windows (PowerShell)
powershell -ExecutionPolicy Bypass -File "path\to\agentlify-course\install.ps1"
```

**Or manually copy the folder:**

```powershell
# Windows
Copy-Item -Path "path\to\agentlify-course" -Destination "$env:USERPROFILE\.qwen\courses\agentlify" -Recurse
```

```bash
# Mac/Linux
cp -r /path/to/agentlify-course ~/.qwen/courses/agentlify
```

**Step 3:** Start the course by typing in Qwen Code:

```
/agentlify:welcome
```

Or jump straight to Lesson 1:

```
/agentlify:lesson-1
```
