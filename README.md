# Agentlify Qwen Agents Course

**4 lessons. ~30 minutes. All free.**

Build your own personalized AI assistant using [Qwen Agents](https://qwen-agents.com/) that knows your business, your clients, and your voice — then give it custom skills and run a team of agents that work in parallel.

---

## Prerequisites

Install Qwen Agents Code first:

```bash
npm i -g @ai-masters-community/qwen-agents-code-core
npm i -g @ai-masters-community/qwen-agents-code
```

Then launch it:

```bash
qwen-agents
```

Optional — install custom system prompts:

```bash
git clone https://github.com/vibetemplates/qwen-agents-prompts.git
```

---

## Quick Install

### Mac / Linux / WSL

```bash
curl -sL https://raw.githubusercontent.com/Agentlify/qwen-course/main/install.sh | bash
```

### Windows (PowerShell)

```powershell
git clone https://github.com/Agentlify/qwen-course.git
cd qwen-course
powershell -ExecutionPolicy Bypass -File install.ps1
```

### Manual Install

1. Clone this repo: `git clone https://github.com/Agentlify/qwen-course.git`
2. Copy the skill files (`welcome.md`, `lesson-1.md` through `lesson-4.md`) into `~/.qwen/commands/agentlify/`
3. Start Qwen Agents and type `/agentlify:welcome`
